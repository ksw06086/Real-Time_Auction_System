# Real-Time_Auction_System
Front는 Nunjucks Template, Back-end는 Node.js/Express 기반의 실시간 경매 프로젝트입니다.

### Using Language
* JavaScript, HTML, CSS

### Using DB
* MySQL(유저, 게시글), MongoDB(경매 메시지)

### Using Library
* Socket.io, SSE(서버센트 이벤트)
